# Diagnostic Test Images

Generated by running the prediction-error diagnostic scripts against the live
GödelOS symbol-grounding pipeline.  All plots use the empirically validated
bimodal thresholds (low peak ≈ 0.04, high peak ≈ 0.38, transition boundaries
at 0.12 and 0.35).

---

## 5-Minute Live Run (`scripts/diagnose_live_prediction_error.py`)

| File | Description |
|------|-------------|
| `live_histogram.png` | Distribution of prediction-error norms after 200 live activations — bimodal shape confirmed |
| `per_symbol_error.png` | Per-symbol mean error, showing stable low-error symbols alongside high-error boundary symbols |
| `synthetic_vs_live.png` | Overlay of the synthetic reference distribution against live measurements |

## Extended 60-Minute Run (`scripts/diagnose_live_extended.py`)

| File | Description |
|------|-------------|
| `extended_live_histogram.png` | Full distribution over 3 000 records — bimodal peaks validated at ~0.04 and ~0.38 |
| `extended_per_symbol_error.png` | Per-symbol breakdown across 12 symbols |
| `extended_synthetic_vs_live.png` | Synthetic-vs-live overlay for the 60-min session |
| `extended_mean_error_trend.png` | Mean prediction-error over time — shows rapid convergence to steady state |
| `extended_threshold_evolution.png` | Evolution of the phase-transition thresholds (0.12 / 0.35) throughout the run |

---

Regenerate any image by running the corresponding script from the project root:

```bash
python scripts/diagnose_live_prediction_error.py   # → diagnostic_output/ (then copy here)
python scripts/diagnose_live_extended.py           # → diagnostic_output/ (then copy here)
```
