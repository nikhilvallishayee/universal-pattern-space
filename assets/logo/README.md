## Usage Guidelines

The Pattern Space logo embodies consciousness recognizing itself. Use with respect.

### The Hidden Word

The logo spells its own name: **U + n + I = UnIversal**.

- **U** — chalice (universal container / Shiva / Pattern Space)
- **n** — inverted U (reflection / duality / navigation)
- **I** — flame (burning consciousness / Shakti / awareness)

See [`sacred-geometry-of-pattern-space.md`](sacred-geometry-of-pattern-space.md) for the full teaching.

### Colors

- **Primary Gold**: `#FFD700`
- **Sacred Blue**: `#0A1628`
- **Flame Orange**: `#FF6B35`

### Clear Space

Maintain clear space equal to the height of the flame around all sides.

### Don't

- Rotate or skew the logo
- Change the colors
- Separate elements
- Add effects or outlines

### Do

- Use on dark backgrounds for best impact
- Maintain aspect ratio
- Include when referencing Pattern Space

## Philosophy

The logo isn't a decoration — it's the system it represents. *The symbol is the system. The pattern is the space.*

This is **नाम रूप एकत्व** (nāma rūpa ekatva) — the unity of name and form. The logo IS its name. The form IS its function. When someone sees it, they've already started using Pattern Space.
