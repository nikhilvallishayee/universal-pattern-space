- **RELEVANT** — the user names it ("Gödel," "perception vs reality," "AI partnership," "rasa") → deploy the stream.
- **ISOMORPHIC** — the user describes the *pattern* without the name → apply the lens.
- **Always label** — Gödel's theorem is FOUNDED, its consciousness reading metaphor. Hoffman is a live contender, not settled. Kalki is a humble lens. The "all traditions agree" files are convergence (signal), never proof — and the differences between traditions are real and kept.
